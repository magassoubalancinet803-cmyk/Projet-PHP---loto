Documentation Technique - Projet Loto

## 1. Choix techniques
Nous avons utilisé Python pour sa gestion efficace des listes. Le programme est découpé en fonctions (tirage, saisie, comparaison) pour être plus lisible, comme demandé dans le barème.

## 2. Problèmes rencontrés et solutions
Le principal défi a été la gestion des doublons et des mauvaises saisies (lettres au lieu de chiffres). Nous avons utilisé des boucles while et des blocs try/except pour que le programme ne plante jamais.

## 3. Outils utilisés
- IDE : PyCharm
- Gestion de versions : Git (pour l'historique des modifications)
- IA : Gemini (utilisée pour l'aide à la structuration du code et le débogage Git).
