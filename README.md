# Projet de Programmation – Jeu de Loto

## Description
Application Python simulant un tirage de loto de 10 numéros (1-49).

## Fonctionnalités
- Génération aléatoire de 10 numéros sans doublons.
- Saisie utilisateur sécurisée avec gestion des erreurs.
- Comparaison des résultats et calcul du score.
- Système de récompense avec bonus si l'ordre est correct.

## Comment lancer le programme
Exécuter le fichier loto.py dans un environnement Python 3
